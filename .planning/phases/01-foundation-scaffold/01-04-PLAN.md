<objective>
Wire Hilt dependency injection at the application level and install placeholder DI modules in all 6 core modules. This establishes the full Hilt graph structure that feature modules will inject from in subsequent phases.

Purpose: SCAF-04 requires Hilt to be "available in all modules." This means the Application class is annotated with @HiltAndroidApp, MainActivity with @AndroidEntryPoint, and each core module provides an @InstallIn module object so its future bindings are discoverable by the graph. Without this wiring, feature ViewModels cannot use @Inject in Phase 2.

Output: Updated AICompanionApp.kt (@HiltAndroidApp + Timber.plant), updated MainActivity.kt (@AndroidEntryPoint), 6 core DI module files (all placeholder objects).
</objective>
